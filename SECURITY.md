# Security Policy

## Supported Versions

The latest version on the `main` branch receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.
Instead, report it privately by emailing **ilyesstaili@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- The potential impact

You can expect an initial response within 72 hours. Responsible disclosure is
greatly appreciated, and contributors will be credited (with consent) once a fix
is released.
